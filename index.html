Telegram üçün avtomatik mesaj kodu

Bu kod vasitəsi ilə telegram qruplarına öz hesabınızdan və ya fake hesabınızdan avtomatik mesaj göndərə bilərsiniz. Bu kodu Pydroid3 və ya termuxda işlədə bilərsiniz amma işlətməzdən əvvəl faylda dəyişikliklər etməlisiniz

1. my.telegram.org burdan api id ve hash alın bu sayta giriş edib və kodda yerinə yazın
2. Message yerinə dırnağın içinə göndərmək istediyiniz mesajı yazın
3. Groups yerinə istədiyiniz qədər qrup tagları yazın mesaj bu qruplara göndəriləcək

Modulları yükləmə:

Termux:
pkg install python
pip install telethon

Pydroid3:
pip install telethon

Bütün bunları etdikdən sonra kodu işlədinin və telegram hesabına giriş edin

Diqqət!!! Bot token or telephone number yazılan yerə bot token yazmayın
