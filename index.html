Telegram üçün avtomatik mesaj kodu

Bu kod vasitəsi ilə telegram qruplarına öz hesabınızdan və ya fake hesabınızdan avtomatik mesaj göndərə bilərsiniz. Bu kodu Pydroid3 və ya termuxda işlədə bilərsiniz amma işlətməzdən əvvəl faylda dəyişikliklər etməlisiniz

1. my.telegram.org burdan api id ve hash alın bu sayta giriş edib və kodda yerinə yazın
2. Message yerinə dırnağın içinə göndərmək istediyiniz mesajı yazın
3. Groups yerinə istədiyiniz qədər qrup tagları yazın mesaj bu qruplara göndəriləcək

Modulları yükləmə:

Termux:
pkg install python
pip install telethon

Pydroid3:
pip install telethon

Bütün bunları etdikdən sonra kodu işlədinin və telegram hesabına giriş edin

Diqqət!!! Bot token or telephone number yazılan yerə bot token yazmayın

Toxun və kodu gör

Esp8266 Wifi Deauther

1. https://deauther.com/docs/download/ burdan öz esp8266 modelinizə uyğun .bin faylını yükləyin.

2. https://esp.huhn.me/ burdan bin faylını esp8266-nıza yükləyin (yalnız pcdən etmək olur).

3. Esp8266-ya elektrik verin və pwned adlı wifi yaranacaq ona qoşulun (parol: deauther).

4. Brauzerə həmin wifinin ipsini yazıb web interfeysə daxil olun.

5. Istədiyiniz wifini seçib deauth hücumunu başladın.

Kali NetHunter rootsuz

Diqqət!! Bunu yükləmək ücün yaddaşınızda ən az 15 gb boş yer olmalıdır və termuxun googledən yüklənmiş yeni versiyası olmalıdır

1. Termuxa apt update && apt upgrade yazıb termuxu güncəlləyin
2. pkg install wget yazaraq wgeti yükləyin
3. wget -O install-nethunter-termux https://offs.ec/2MceZWr
4. chmod +x install-nethunter-termux
5. ./install-nethunter-termux yazdıqdan sonra 1- seçib enter vurun və biraz gözləyin 2 gb lıq fayl yüklənəcək.
6. Fayl yükləndikdən sonra yenə ./install-nethunter-termux yazıb 1 seçin sonra isə n seçin və təxmini 10 dəqiqə gözləyib yenə n seçin
7. mkdir chroot
8. mv kali-arm64/ chroot/
9. Yenə ./install-nethunter-termux yazıb 1 seçin və digər seçimlərdə n seçin.
10. Təbriklər siz kali nethunteri yüklədiniz termuxda nh yazaraq nethunterə girə bilərsiniz.

Sudonun parolu: kali
